This project focuses on classifying software requirement statements into specific quality attributes such as **Performance**, **Usability**, **Reliability**, **Security**, etc., using **Natural Language Processing (NLP)** and **Machine Learning** techniques.

3. **Model Training** ‚Äì Applying ML algorithms such as:
   - Random Forest  
   - Logistic Regression

5. **Visualization** ‚Äì Displaying accuracy and confusion matrix for each classifier.  
6. **Testing** ‚Äì Predicting category for new input requirement statements.
